# Test Plan：notebooklm-learn-new-domain

## 測試策略

本專案交付物是文件（提問模板 + 使用手冊 + SKILL.md），不是程式碼。測試方式為手動驗證，分三個層級：

1. **內容驗證** — 文件本身是否正確、完整
2. **功能驗證** — 提問模板在 NotebookLM 中是否有效
3. **體驗驗證** — 零基礎用戶能否順利使用
